# Lambda Ledge

Lambda Ledge is a 2D-platformer, where the player's goal is simple: **get to the exit**

## Installation 
Clone the repository and `cd` into it. Run `make start` to begin playing. 

## Controls

<center>

| Gameplay   | Keys           |
|------------|----------------|
| Move Left  |  ←             |
| Move Right |  ←             |
| Move Up    |  ↑             |
| Move Down  |  ↓             |
| Jump       | <kbd>c</kbd>   |
| Dash       | <kbd>x</kbd>   |
| Quit       | <kbd>q</kbd>   |

</center>
